# 🎮 ZS3 Memory Game

Gra memory stworzona na Dni Otwarte ZS3 Ruda Śląska 2026.

## 📋 Opis
Dopasuj pary obrazków związanych z kierunkami szkoły — Informatyk, Elektryk i Spedytor.

## 🎯 Jak grać
1. Wpisz swoje imię
2. Wybierz poziom trudności (4x4, 6x6, 8x8)
3. Klikaj karty i dopasowuj pary
4. Wygraj przed upływem czasu!
